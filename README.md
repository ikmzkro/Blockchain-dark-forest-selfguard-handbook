### 📊 **6. マルチシグ・ERC2771・ERC4337の比較**
**目的:** 各方式のメリット・デメリットを理解し、要件に応じた選択を行う  

| 項目           | マルチシグ                          | ERC2771 (Meta-TX)               | ERC4337 (AA)                    |
|----------------|-------------------------------------|--------------------------------|--------------------------------|
| **鍵管理**     | 複数の鍵署名が必要                   | フォワーダー経由の署名           | バンドラー経由のアカウント管理 |
| **ガス代負担** | 送信者負担                          | フォワーダーまたはdApp負担       | Paymaster経由で第三者負担       |
| **セキュリティ**| 高（複数署名）                      | 中（1署名）                     | 高（アカウント抽象化とMPC）     |
| **実装難易度**| 中（マルチシグコントラクト構築が必要）| 低（フォワーダーを設定するだけ） | 高（ERC4337インフラ構築が必要） |
