## a0（最初の係数）が秘密情報そのものであるというのは、直感的には少し気持ち悪い
これは、シャミアの秘密分散法の特性によるもので、秘密情報を多項式の定数項として扱うことで、他の係数と組み合わせて秘密を分散させることができます。他のランダムな係数と組み合わせることで、秘密を保護することができる点です。これにより、閾値以上のシェアが集まらない限り、元の秘密を復元することができません
```
shamirsecret: ShamirSecret {
  threshold: 2,
  secret: <Buffer 49 6e 20 74 68 65 20 6e 61 6d 65 20 6f 66 20 41 64 69 20 53 68 61 6d 69 72>,
  coefficients: [
    <Buffer 49 6e 20 74 68 65 20 6e 61 6d 65 20 6f 66 20 41 64 69 20 53 68 61 6d 69 72>,
    <Buffer 60 50 a1 6c e2 cd f6 35 a3 c8 16 67 75 06 cf 25 40 5d fd a9 1d c5 f0 aa 34>
  ]
}
```

## threshold: 1の場合は復元もクソもないのでは
秘密情報は1つのシェアに完全に依存しているため、他のシェアが必要ない状態になるのでその通り。
```
shamirsecret: ShamirSecret {
  threshold: 1,
  secret: <Buffer 49 6e 20 74 68 65 20 6e 61 6d 65 20 6f 66 20 41 64 69 20 53 68 61 6d 69 72>,
  coefficients: [
    <Buffer 49 6e 20 74 68 65 20 6e 61 6d 65 20 6f 66 20 41 64 69 20 53 68 61 6d 69 72>
  ]
}
```
